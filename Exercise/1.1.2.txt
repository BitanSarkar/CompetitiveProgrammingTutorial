1 <= N <= 8
n = 2N
T(n) = n^2*T(n-2)

T(n) = n(n-1)/2*T(n-2)

Thus,
T(n) = n^2*(n-2)^2...2^2 = 4^N*N!
So O(N) = 4^N*N!

T(n) = (n(n-1) / 2) * ((n-2)(n-3) / 2) * ... * (2*1/2) = (2N)!/2^N
